# B2B 통합 물류 관리 시스템

## 물류관리프로젝트

이 프로젝트는 B2B 통합 물류 관리 시스템을 개발하는 것입니다.

### 데모 링크
[여기를 클릭하십시오](http://131.186.18.42/)

### 제작 기간 & 참여 인원
이 프로젝트는 개인 프로젝트로 진행되며, 제작 기간은 2주입니다(진행중).

### 사용한 기술
* Spring Boot 3.0.4
* Spring Security 6.0.2
* Thymeleaf 3.1.1
* Mysql 8.0.32
* JPA 6.0.6
* QueryDSL 5.0.0
* Oracle Cloud
* Cent OS 8
* Jenkins
